# Demo 6: React-asiakassovelluksen toteutus - Ostoslista-sovellus

Nyt pääsemme yhdistämään React-sovellukset ja REST API:n mukaiset Web Service palvelinsovellukset yhdeksi kokonaisuudeksi. Demossa 6 aiempien demojen Ostoslista-sovellus on nyt toteutettu graafisena ns. asiakassovelluksena (client application).

Demossa iso osa koodista perustuu jo aiempiin demoihin, eikä niitä käydä uudelleen läpi. Demossa asiakassovellus on toteutettu Create React App -työkalun sijaan Vite:llä, joka on nykyaikainen standardi React-sovellusten toteuttamiseen ja tästä sisällytetään pieni kertaus/tiivistelmä toteutuksesta. Olet saattanut käyttää aiemmalla Sovellusohjelmointi 1 -toteutuksella CRA:ta React-sovellusten tekemiseen, mutta tätä ei ole enää tuettu aikoihin, joten on tarkoituksenmukaisempaa siirtyä nykyaikaisempiin tekniikoihin. Vite ei kuitenkaan poikkea merkittävästi React-sovelluksena CRA:sta.
