Demo 7: JWT-autorisointi

Tämä on demo REST API -palvelinsovelluksen React-asiakassovelluksen yhteistoiminnasta. Lisäyksenä edelliseen demoon on JWT (JSON Web Token) -autorisointi. JWT-autorisoinnilla voidaan varmistaa, että palvelinsovellusta voi käyttää ainoastaan niistä asiakassovelluksista, joiden kutsuisssa on asianmukainen token. Käynnistä Reactin kehityspalvelin suorittamalla komento 'npm start' client-kansiossa

Testaa asiakassovelluksen toiminnallisuutta selaimella osoitteessa http://localhost:3000. Palvelinsovellusta voit testata Postman-ohjelmalla osoitteessa http://localhost:3007/api/ostokset, jos laitat tokenin osaksi pyyntöjä headers-välilehdellä (Authorization) samalla tavoin kuin React-sovelluksessa
